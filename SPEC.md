# 健診結果の見方【無料】血液検査・健康診断の数値を解説 - 技術仕様書

## 概要

**サービス名**: Health Check Explainer
**バージョン**: 1.1.0 (2026-06-03)
**更新日**: 2026-06-03
**URL**: https://appadaycreator.com/health-check-explainer/

健康診断・血液検査の結果数値の意味を分かりやすく解説。気になる項目の改善方法も提案。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 新機能（v1.1.0）

### 1. スマートフォン最適化入力フロー
- 各フィールドに正常範囲を表示
- `inputmode` 属性で数値キーボード自動表示
- タッチターゲット 48px 以上確保

### 2. 計測履歴管理
- localStorage で最新 20 件の履歴を自動保存
- 「計測履歴を見る」ボタンで過去の測定値を確認
- 日付付きで履歴を管理

### 3. 結果のシェア機能
- 📋 **結果をコピー**: 診断結果をテキスト化してクリップボードにコピー
- 𝕏 **Twitter シェア**: 結果と URL を自動入力して共有
- **LINE シェア**: LINEで友人・医師と共有

### 4. 動的な推奨グッズ表示
- 異常値に基づいて Amazon リンクを動的に表示
  - 高血圧 → 血圧計を推奨
  - 高血糖 → 血糖測定器を推奨
  - 高コレステロール → DHA/EPA サプリを推奨
  - 肥満 → 体重管理グッズを推奨

### 5. 関連ツール・サービスの適切化
- 不適切な「老後資金」「ふるさと納税」「ペット選び」を削除
- 健診関連ツール「BMI 管理」「サプリメント診断」「メンタルヘルス」「睡眠管理」に統一

## 使い方

1. ページを開き、入力フォームの項目を確認する（正常範囲を参考に）
2. 必要な情報を入力する（自動数値キーボード表示）
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じて以下を活用:
   - 📋 結果をコピー（テキスト化）
   - 𝕏/LINE でシェア（SNS共有）
   - 📊 計測履歴から過去の測定値を確認

## よくある質問（FAQ）

**Q: 健診結果の見方は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
